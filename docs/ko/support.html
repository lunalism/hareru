고객지원

Q1. "본당의 지출"이란 무엇인가요?

내부 이체(내 통장→내 통장)를 자동으로 제외하고, 실제로 쓴 돈만 표시하는 Hareru의 핵심 기능입니다.

Q2. 은행 계좌 연동이 되나요?

현재 은행·카드 자동 연동은 지원하지 않습니다. 모든 기록은 수동 입력입니다.

Q3. 데이터는 어디에 저장되나요?

모든 데이터는 사용자의 기기 내에만 저장됩니다. 외부 서버로 전송되지 않습니다.

Q4. 무료로 사용할 수 있는 기능은?

지출·수입·이체 기록, 내부 이체 자동 제외, 예산 관리, 카테고리별 리포트, 금융 용어 사전이 무료입니다.

Q5. 지원 언어는?

일본어, 한국어, 영어 3개 언어를 지원합니다. 시스템 언어에 따라 자동으로 표시됩니다.

Q6. 데이터를 삭제하려면?

설정 화면에서 "데이터 삭제"를 선택하면 모든 데이터가 삭제됩니다. 앱을 삭제해도 기기 내 데이터가 제거됩니다.
